# White Blood cell segmentation

## Dataset
Kaggle Dataset:
https://www.kaggle.com/datasets/inhvnnhn/krd-wbc-dataset

## How to Run
1. Open a colab/kaggle notebook
2. Click "Add Data" using the link which i have provided
3. Add the dataset
4. Run all cells

## Metrics
- Dice Score
- Jaccard Index
- Hausdorff Distance
- Boundary F1-Score
