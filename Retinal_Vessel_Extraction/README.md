# Retinal Vessel Extraction using Niblack & Sauvola

## Dataset
Kaggle Dataset:
https://www.kaggle.com/datasets/andrewmvd/drive-digital-retinal-images-for-vessel-extraction

## How to Run
1. Open a colab/kaggle notebook
2. Click "Add Data" using the link which i have provided
3. Add the dataset
4. Run all cells

## Metrics
- Sensitivity:

  Sensitivity = TP / (TP + FN)

  Evaluation is performed specifically on thin vessel pixels.
